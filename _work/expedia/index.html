Expedia Exercise

Assumptions

  • Used an existing company's color palette
  • Ignored navigation
  • Flow represents a returning user who has ordered items from these stores before.
  • The user has ordered items from two separate stores: Whole Foods and Fred Meyer.

User Stories

As a customer who's just submitted my order for a grocery delivery, I want to:

  • stay updated on the status of my order(s) so I know when to expect the delivery.
  • add last-minute items so I can get an item I forgot initially.
  • contact my Shopper in case I need to relay information about my order.
  • contact Company support in case I have an issue with my order or the app.
  • save frequently purchased items so I can easily reorder them the next time I shop.
  • cancel my order if I changed my mind.


Process

After writing user stories, I sketched some initial ideas for the flow.
