Cruise Ship Migration

Existing cruise ship pages were disconnected, dated, and lacked high-level features that consumers need.

Existing Design

Users

Travel advisors and consumers researching cruises on our website

User's Goal

Find the key features of the ship and find sailings available on the vessel

My Approach

  • Before, the user had to leave our website to view the ship content since it's externally hosted
  • Use existing analytics to find users' most needed content and make sure it is easy to find
  • Modernize the look and feel, and create a stronger tie to The University of Texas at Austin brand
