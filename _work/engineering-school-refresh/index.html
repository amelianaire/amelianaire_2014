Engineering School Website CSS Refresh

While awaiting a full responsive redesign, our school's site needed a facelift using minor CSS updates.

Before

Our goals for the refresh:

  • Reduce the confusing, rarely clicked audience navigation (top right)
  • Reformat the carousel to hold larger images
  • Simplify and clean up the overall design, providing clearer hierarchy
  • Add flexible spaces for new types of content

After

  • By slightly reformatting key sections of the home page, hierarchy is clearer
  • New flex space underneath the "news" section provides easy customization options
  • An expanded footer shows stronger alignment with the UT Austin brand while providing emphasis on social media
